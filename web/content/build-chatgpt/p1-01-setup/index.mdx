  <Step title="The Toy Dataset">
    <Description>
      ## String Reversal
      
      We don't need massive GPUs to learn the architecture. We will create a synthetic dataset: **String Reversal**.
      
      - Input: `a b c`
      - Output: `c b a`
      
      This task forces the model to learn long-range dependencies and memory.
      
      ### Your Task
      Write a `Dataset` class that generates random sequences of characters and their reversals.
    </Description>
