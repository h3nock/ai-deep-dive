    <Description>
      ## Assembly Time
      
      Now we assemble the components we learned in Steps 1-8.
      
      ### 1. The Encoder
      - **Input**: Source sequence (e.g., "a b c").
      - **Layers**: Self-Attention (Unmasked) -> FFN.
      - **Output**: A sequence of context vectors.
      
      ### 2. The Decoder
      - **Input**: Target sequence (shifted right).
      - **Layers**: Masked Self-Attention -> **Cross-Attention** -> FFN.
      - **Output**: Next token probabilities.
      
      ### Your Task
      Implement the `Transformer` class that combines the Encoder and Decoder.
    </Description>
