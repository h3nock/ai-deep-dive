description: "How all components combine into the repeatable transformer block."
---

  The GPT architecture is built from a single repeating unit called the Transformer block. Each block arranges the components from the previous chapters into a specific internal pattern, and every layer in the model uses that same pattern with its own learned weights. The complete model stacks these blocks between input processing that converts token IDs to vectors and output processing that converts vectors to predictions.
</Description>
